projects

4 U 2 Write

[HTML, CSS, JS, REACT, FIREBASE]

A daily writing tool to get focused work done. Users can choose an amount of time to write for. If they are not typing for more than 15 seconds while the timer is in session, there is a warning to return the user to the task of writing. Lastly, each day the app produces a new communal writing prompt for all users to draw inspiration from, and users can add a prompt to the list of prompts (stored in Firebase).

Final Fantasy XIV Gear Planner

[HTML, CSS, JS, REACT, FIREBASE]

A tool for groups playing Final Fantasy XIV to plan gear distribution. Users can create a character, choose which gear items they would want, and store its details in Firebase to edit later. When a group gets a new item, the app compares the item to each character's profile to determine who could receive it.
